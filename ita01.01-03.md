# ita-exercises 1.1-3

## Elige una estructura de datos conocida y comenta sus fortalezas y sus debilidades

  * Vector (array) permite el almacenamiento continuo de datos.
  * Fortalezas:
    * Permite acceso aleatorio a los elementos.
	* Mínimo coste de almacenamiento comparado con las listas encadenadas.
	* Coste de tiempo constante para acceder al elemento n.
  * Debilidades:
    * Coste superior para el acceso secuencial a todos los elementos frente a una lista.
	* Las inserciones y borrados requieren el desplazamiento del resto de los elementos.
	* Hay que predefinir el tamaño del array.
